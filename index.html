The Platform

What no other
platform offers

INTELLEA's innovation lies in its multidimensional intelligence integration. While commercial platforms demonstrate significant capability in one or two dimensions, none integrate configurable Socratic pedagogy, multiagent specialist grading, real time tri modal emotion recognition, and a semantically unified Block Code IDE within a single teacher controlled architecture.

The separation of instructor intent from student state is a key design principle: the configured style governs the mode of interaction while BKT mastery estimates and emotional analysis influence the intensity and scaffolding within that style.

01 Core Innovation

Configurable Teacher Persona System

Rather than enforcing a single interaction paradigm, the system exposes four pedagogically grounded teaching styles with continuous scalar parameters, producing an effectively unlimited persona space.

A supervisor agent validates every generated response for style fidelity through a cyclic review mechanism, ensuring pedagogical constraints are maintained across all interactions.

02 Core Innovation

Tri Modal Emotion Fusion

Single modality systems suffer from well documented limitations. Facial analysis is vulnerable to occlusion, speech systems are sensitive to noise, and text systems miss prosodic cues.

INTELLEA's Feed Forward Neural Network learns the weighted importance of each modality alongside confidence scores, dynamically compensating when individual modalities are unreliable.

03 Core Innovation

SAGE Assessment Engine

SAGE introduces several novel assessment mechanisms that address gaps identified in current literature and mirror academic conventions that prior automated systems have ignored.

Novel

Follow Through Marking

Awards partial credit when students apply correct reasoning from incorrect intermediate results.

Novel

Reference Validated Test Cases

LLM generated test cases validated against reference solutions eliminate hallucinated ground truth.

Novel

Harness Injection

Function level automated assessment across multiple programming languages.

Report

04 Core Innovation

Hybrid Block Code IDE

The single AST backed deterministic translation ensures semantic equivalence between visual block programming and traditional code. Submissions in either modality remain analytically comparable through explicit coupling with both the SAGE assessment engine and the ATLAS tutoring system.

  • Deterministic bidirectional translation between blocks and code
  • Single Abstract Syntax Tree ensures semantic equivalence
  • Coupled with SAGE and ATLAS for assessment and tutoring

Research Foundations

Grounded in
learning science

The operationalization of Cognitive Constructivist Learning Theories through configurable agentic frameworks translates established pedagogical theory into computational architecture.

BKT

Bayesian Knowledge Tracing

Probabilistic model estimating student mastery using prior knowledge, learn rate, guess and slip parameters to track knowledge state evolution.

P(Lₙ) = P(Lₙ₋₁|obs) + (1 − P(Lₙ₋₁|obs)) · P(T)
ZPD

Zone of Proximal Development

Vygotsky's ZPD implemented as a computable score, selecting content that is "just challenging enough" for optimal growth within each learner's current boundary.

RAG

Socratic Strategies + RAG

Collins and Stevens' Socratic strategies formalized as LLM prompt constraints, with all outputs grounded in course specific knowledge through Retrieval Augmented Generation.
