01 · LITERATURE SURVEY

A century of educational psychology
meets multi-agent AI

INTELLEA is grounded in established literature on intelligent tutoring, knowledge tracing, multi-agent orchestration, multimodal emotion recognition, and dual-modality programming environments.

Tutoring

Intelligent Tutoring Systems

VanLehn (2011); Kulik & Fletcher (2016)

Meta-analyses establishing that one-on-one ITS approach human-tutor effectiveness, with effect sizes around d = 0.76 over conventional instruction.

Knowledge Tracing

Bayesian Knowledge Tracing

Corbett & Anderson (1995)

Probabilistic model estimating mastery via prior, learn-rate, guess and slip parameters; INTELLEA uses BKT to drive ZPD-aligned content selection.

Orchestration

Multi-Agent Frameworks

LangGraph · CrewAI · AutoGen

Comparative analysis of agentic frameworks. LangGraph's stateful DAG with self-loops powers our supervisor-validated tutoring and grading workflows.

Retrieval

RAG for Education

Lewis et al. (2020); Cogniti

Retrieval-Augmented Generation grounds LLM outputs in course materials, mitigating hallucination; essential for trustworthy educational content.

Emotion

Multimodal Emotion Recognition

D'Mello & Graesser; Poria et al.

Affective computing in learning: face, voice, and text fusion improves recognition robustness when single modalities suffer from noise or occlusion.

Assessment

Automated Grading

Burrows et al.; Zhang et al.

Existing systems handle objective items well but struggle with subjective and code grading. SAGE addresses this via specialist multi-agent evaluators.

Programming

Block-based Environments

Scratch · Snap! · Blockly · MakeCode

Visual programming reduces syntactic friction. INTELLEA extends this with deterministic AST-backed translation to textual languages.

Reference

Reference Systems

Cogniti · Squirrel AI · Knewton · vNMF · FACET

Each demonstrates strong capability on one or two dimensions; none integrate all six dimensions INTELLEA targets in a single architecture.

Key Gap Statement

No existing system consolidates adaptive tutoring, configurable pedagogical styles, multi-agent automated grading, real-time emotion detection, dual-modality programming, and RAG-grounded content delivery within a single teacher-controlled platform.

03 · RESEARCH PROBLEM

Fragmentation reduces
pedagogical coherence

Current AI-powered educational platforms exhibit fragmentation. Emotional intelligence, automated assessment, coding environments, and adaptive difficulty algorithms exist in isolation; each developed by different teams, with different assumptions, and no shared state.

Teachers must juggle multiple disconnected tools that were not designed for state sharing. A frustrated student detected in one tool does not soften the assessment style in another. A misconception identified during tutoring does not surface during grading. The cognitive load on teachers, and the loss of pedagogical coherence, is significant.

INTELLEA's contribution is not any single component, but the integration: a single architecture where mastery state, emotional state, and pedagogical configuration flow between tutoring, assessment, coding, and content delivery.

04 · OBJECTIVES

Five objectives,
one integrated platform

01

Multi-agent adaptive tutoring (ATLAS)

Design an ATLAS subsystem with four configurable teaching styles, BKT-driven mastery tracking, ZPD-aligned content selection, and RAG-grounded delivery.

02

Multi-agent automated grading (SAGE)

Develop a SAGE engine with specialist evaluators for objective, subjective, mathematical, and coding submissions; with self-correction and confidence-gated review.

03

Hybrid Block–Code IDE

Build a Hybrid IDE with a shared AST enabling deterministic bidirectional translation between visual blocks and textual programming languages.

04

Multimodal emotion recognition (VIBE)

Integrate a VIBE subsystem fusing facial, speech, and textual signals via FFNN to enable emotion-aware adaptive teaching.

05

Controlled pilot study evaluation

Evaluate the integrated system through a controlled pilot measuring learning outcomes, knowledge retention, grading accuracy, and pedagogical fidelity. Pilot N = 62 secondary-school students; treatment-vs-control with 4-week retention follow-up.

05 · METHODOLOGY

Four subsystems,
one shared state

INTELLEA's architecture is composed of four integrated subsystems sharing mastery state, emotional state, and pedagogical configuration through LangGraph-orchestrated workflows.

Subsystem 01 ATLAS Adaptive Tutoring & Learning Architecture System
  • LangGraph Virtual Teacher: multi-agent workflow with 9 specialist nodes
  • Four teaching styles: Socratic, Direct Instruction, Collaborative, Scaffolded
  • BKT mastery tracking with ZPD-based content selection
  • ACE engine: 7 question types aligned to Bloom's Taxonomy with 3-validator pipeline
  • RAG pipeline: Pinecone vector store, OpenAI embeddings
  • ElevenLabs voice synthesis with emotion-modulated parameters
Subsystem 02 SAGE Specialist Assessment & Grading Engine
  • 5 specialist evaluator agents: Objective, Subjective, Math, Coding, Validator
  • LangGraph DAG orchestration with self-correction up to 3 iterations
  • Confidence-gated mandatory teacher review for borderline cases
  • TF-IDF plagiarism detection with cosine similarity threshold 0.85
  • Automated study recommendations for scores below 60%
  • Gamified badge system: 4 tiers × 5 categories
Subsystem 03 Hybrid IDE Block ↔ Code Programming Environment
  • Shared AST in normalized JSON schema
  • Bidirectional translation: Python, Java, C++, React
  • Sandboxed execution with WebSocket-based interactive I/O
  • PDF-to-Coding lesson generator pipeline
  • Adaptive difficulty service with rule-based challenge selection
  • Block recommendation with Socratic hints
Subsystem 04 VIBE Visual-Interactive Behavioral Emotion detection
  • Speech: MFCC features, deep model on RAVDESS, CREMA-D, TESS, SAVEE; 73% acc
  • Video: Frame sequences, CLSTM on CREMA-D; 71% acc
  • Text: GloVe embeddings, Bi-LSTM; 91.5% acc
  • Fusion: FFNN combining predictions and confidence scores
  • Adaptation: emotion drives ATLAS scaffolding intensity, hint strategy, voice parameters
